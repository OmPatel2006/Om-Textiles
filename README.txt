To access my product you will have to download XAMPP as my product includes PHP files and needs XAMPP to run. Also for the database to work you will have to import the SQL file in phpmyadmin after you download XAMPP. After you have installed XAMPP and imported the SQL file than you have to copy the "Om Textiles" folder and paste it in the "htdocs" folder in XAMPP files. Then to open the website run the XAMPP control panel start Apache and MySQL. Then open your browser and type "http://localhost/Om%20textiles/signup.html" to lauch the website and start from signup page.
